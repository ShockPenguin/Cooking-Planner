Update PROGRESS.md to reflect the work just completed.

## Steps

1. Read the current `PROGRESS.md`.
2. Under `## Done`, add a checked item for what was just finished:
   `- [x] YYYY-MM-DD — <what was done, one line>`
   Use today's date.
3. If new tasks were discovered, add them as unchecked items under `## Next`.
4. If an approach was tried and failed, add a one-line note under `## Failed Approaches` explaining why it didn't work (so it isn't re-attempted).
5. Remove anything from `## Next` that is now done.
6. Write the updated file.
7. Print: `PROGRESS.md updated.`

Keep all entries to one line. The goal is a scannable log, not a narrative.
